# Моделирование двухслойного пласта

## Курсовая работа по предмету "Методы математической физики в нефтяном инжиниринге"

Данная программа реализует математическую модель фильтрации флюида в двухслойном пласте при отборе жидкости из скважины. Модель позволяет исследовать нестационарные процессы распределения давления и перетоков между пропластками с различными фильтрационно-емкостными свойствами.

Решение является аналитическим и основано на применении преобразования Лапласа к уравнениям пьезопроводности с последующим численным обращением методом Стефеста. 

В модели учитываются:
- Размеры пластов (радиусы контуров питания)
- Фильтрационные параметры (проницаемость, пористость, мощность)
- Скин-факторы (призабойные зоны)
- Взаимодействие между пластами через общий ствол скважины

### Теоретическая основа

Математическая постановка задачи включает:

1. **Уравнение пьезопроводности** для каждого пласта:
   $$\frac{\partial^2 p_i}{\partial r^2} + \frac{1}{r}\frac{\partial p_i}{\partial r} = \frac{1}{\kappa_i}\frac{\partial p_i}{\partial t}, \quad i = 1,2$$

2. **Начальные условия**:
   $$p_i(r,0) = p_{initial}$$

3. **Граничные условия**:
   - На забое скважины (условие общего дебита):
     $$\sum_{i=1}^2 \frac{2\pi k_i h_i}{\mu} \left(r\frac{\partial p_i}{\partial r}\right)_{r=r_w} = -Q(t)$$
   - На контурах питания:
     $$p_i(r_{ei},t) = p_{initial}$$
   - Условия сопряжения на забое (равенство забойных давлений):
     $$p_{wf1}(t) = p_{wf2}(t) = p_{wf}(t)$$

4. **Учет скин-фактора** (призабойной зоны):
   $$p_{wf}(t) = p_i(r_w,t) - S_i r_w \frac{\partial p_i}{\partial r}\bigg|_{r=r_w}$$

Решение ищется в пространстве изображений Лапласа, где система дифференциальных уравнений сводится к модифицированным уравнениям Бесселя. После определения коэффициентов в пространстве изображений производится численное обращение методом Стефеста для получения оригиналов функций давления и дебитов.


### Программа позволяет:

1. **Рассчитать нестационарное забойное давление** при заданном дебите скважины
2. **Определить распределение дебитов между пластами** в зависимости от времени
3. **Построить профили давления** по радиусу каждого пласта для различных моментов времени
4. **Визуализировать производную давления** по логарифму времени для анализа режимов течения

### Структура программы

#### Класс `TwoLayerReservoir`

Основной класс, реализующий математическую модель двухслойного пласта.

**Атрибуты:**
- `p_initial` — начальное пластовое давление (Па)
- `rw` — радиус скважины (м)
- `mu` — вязкость флюида (Па·с)
- `Q_total` — дебит скважины (м³/с)
- `re1, re2` — радиусы контуров питания (м)
- `k1, k2` — проницаемости пластов (м²)
- `m1, m2` — пористости пластов
- `h1, h2` — мощности пластов (м)
- `S1, S2` — скин-факторы
- `kappa1, kappa2` — пьезопроводности (м²/с)
- `kpr1, kpr2` — гидропроводности (м³·с/кг)

**Методы:**
- `stehfest_invert()` — обращение преобразования Лапласа методом Стефеста
- `get_bessel()` — вычисление модифицированных функций Бесселя
- `get_coefficients()` — решение системы для коэффициентов разложения
- `pressure_disturbance_at_well()` — возмущение давления на забое в пространстве Лапласа
- `flow_rate()` — дебит из пласта в пространстве Лапласа
- `pressure_disturbance_at_radius()` — возмущение давления в точке r в пространстве Лапласа

#### Вспомогательные функции

- `calculate_time_data()` — расчет временных зависимостей (забойное давление, дебиты)
- `calculate_radial_profiles()` — расчет профилей давления по радиусу
- `plot_results()` — визуализация временных зависимостей
- `plot_radial_profiles()` — визуализация профилей давления

### Выходные данные и визуализация

Программа генерирует три основных графика:

1. **Забойное давление от времени** — логарифмический масштаб по времени, отображение начального и установившегося давления
2. **Дебиты пластов от времени** — распределение общего дебита между пропластками
3. **Производная давления по логарифму времени** — диагностический график для идентификации режимов течения

Дополнительно строятся **профили давления по радиусу** для каждого пласта в различные моменты времени (0.1, 1, 10, 100, 1000 часов).
